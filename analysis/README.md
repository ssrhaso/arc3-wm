# analysis/

Evidence artifacts backing the progress report's results tables.

- `p4_fromscratch_vs_proper.md` - paired cold-vs-warm RHAE comparison -> **Table 1** (paired RHAE @ 500k env-steps).
- `p4_cross_cell_diagnosis.md` - the WM-fits / controller-fails dissociation across the 6 pilot warm cells -> **Table 2**. Generalizes the single-cell `p4_vc33_dryrun_diagnosis.md`.
- `p4_vc33_dryrun_diagnosis.md` - the single-cell (vc33) forensic diagnosis the cross-cell doc extends.
- `build_p4_cross_cell_diagnosis.py`, `build_p4_combined.py` - regenerate the markdown diagnoses above. They read local-only run artifacts (`_p4_analysis/`, B2-cached logdirs) and so are not runnable from a fresh clone; the generated `.md` files are committed as the durable record.

`benchmark_table.{md,json}` are generated by `scripts/build_benchmark_table.py` and are gitignored.
